Build instructions

{% for release in site.data.releases offset: 0 limit: 1 %}

To download and build QEMU {{release.branch}}.{{release.patch}}:

wget http://qemu-project.org/download/qemu-{{release.branch}}.{{release.patch}}.tar.bz2
tar xvJf qemu-{{release.branch}}.{{release.patch}}.tar.bz2
cd qemu-{{release.branch}}.{{release.patch}}.tar.bz2
./configure
make
{% endfor %}

To download and build QEMU from git:

git clone git://git.qemu-project.org/qemu.git
cd qemu
git submodule init
git submodule update --recursive
./configure
make

Detailed compilation instructions can be found in the wiki for Linux, Win32 and macOS.
